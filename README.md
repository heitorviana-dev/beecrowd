## 📖 Sobre

Este repositório contém minhas soluções para os problemas do **Beecrowd**, uma plataforma brasileira online para prática de programação competitiva e resolução de problemas algorítmicos.

O objetivo é documentar o progresso, organizar as soluções por categoria e servir como material de estudo e referência.

Os problemas foram utilizados utilizando jupyter notebook, logo a extensão utilizada é ipynb.

## 🚀 Como Usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/heitorviana-dev/beecrowd.git
   cd beecrowd-solutions
   ```

2. **Navegue até a pasta do problema desejado** e abra o arquivo na linguagem de sua preferência.

3. **Para testar localmente**, copie o código, acesse o [Beecrowd](https://www.beecrowd.com.br/), localize o problema pelo ID e submeta.
